# Case 010 — Sleep Intake Forms
*Filed: 2026-04-09 | Bear Country*

**What failed:** Every sleep intake form assumes the answer to "what's keeping you up?" is cognitive arousal — racing thoughts, anxiety, rumination. There is no box for "none of the above." No box for yoga nidra. No box for time blindness. Without the right question, insomnia was invisible — disguised as hypersomnia for years.

**Human cost:** Wrong diagnosis pathway. Wrong treatment. Years.

**Who it fails:** Anyone whose nervous system doesn't run the standard internal monologue.

**oz feature:** Oz knows. Oz has the history. Oz doesn't ask the wrong question.

**Classification:** *Bear Country — the person who wrote the form has self-talk.* No malice. The epistemological gap is genuine. The form builder has never experienced an absence of internal monologue, so they never thought to ask about it.
